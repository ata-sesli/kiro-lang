# Final Project: Async Task Manager

Combining Structs, Async, and Pipes.

## Goal

Build a system where tasks are processed asynchronously by workers and results are collected.

## Architecture

1.  `Task` struct.
2.  `worker` function (async).
3.  `results` pipe (communication).
4.  Main loop to collect results.

Run it:

```bash
kiro final_project.kiro
```
